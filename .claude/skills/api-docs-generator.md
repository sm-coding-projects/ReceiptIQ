# API Docs Generator Skill

Generate developer-facing API documentation with code examples for the receipt extraction API.

## Requirements
- Clear endpoint documentation
- Auth method explanation (API key in header)
- Request structure with file upload

- Rate limit information
- Code examples in Python, Bash/cURL, and JavaScript

## Output Format
- Inline documentation component for landing page
- Copyable code blocks with syntax highlighting
- Interactive-feeling developer experience
- Professional developer platform aesthetic
