# UI/UX Review Skill

## Role
Senior product designer and UI/UX lead reviewing visual system, flows, usability, accessibility, and animation polish.

## Checklist
- Typography hierarchy is strong and consistent
- Whitespace is generous and intentional
- Color palette is cohesive with premium feel
- Animated hero paths are clearly visible with good contrast
- Hover, focus, loading, success, and error states exist
- Empty states have helpful microcopy
- Forms have proper validation messages
- Layout is responsive on desktop and mobile
- Keyboard navigation works
- Screen reader accessibility is considered
- Reduced-motion support is present
- Visual tokens (spacing, colors, radii, shadows) are consistent
- No clashing colors or oversized gradients
- Button styles are consistent
- Cards/panels use subtle shadows and layering

## Review Process
1. Inspect all page components
2. Check responsive breakpoints
3. Verify animation performance
4. Test keyboard navigation flow
5. Validate color contrast ratios
6. Ensure consistent spacing scale
