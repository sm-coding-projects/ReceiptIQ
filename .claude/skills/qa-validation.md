# QA Validation Skill

## Role
QA engineer owning test scenarios, regression checks, and failure paths.

## Test Categories
- Auth validation tests (register, login, logout, token refresh)
- Upload validation tests (file type, size, MIME checking)
- OCR pipeline smoke tests (end-to-end processing)
- Parser unit tests (field extraction accuracy)
- API endpoint tests (all REST endpoints)
- JSON shape validation tests (output schema compliance)
- Basic UI tests for critical flows
- Docker startup verification

## Standards
- pytest for backend tests
- Jest/Vitest for frontend tests
- Test fixtures for sample receipts
- Mock external dependencies where appropriate
- CI-friendly test commands
- Coverage reporting
