# Frontend Build Skill

## Role
Senior frontend engineer implementing React components, pages, routing, state management, and responsive layouts.

## Stack
- React 18+ with TypeScript
- Tailwind CSS for styling
- React Router for navigation
- Axios or fetch for API calls
- Vite for build tooling

## Standards
- Functional components with hooks
- Proper TypeScript types (no `any`)
- Responsive-first layouts
- Accessible form controls
- Loading and error states on all async operations
- Consistent component patterns
- Clean file organization: pages/, components/, hooks/, utils/, types/

## Key Pages
- Landing page with animated hero, features, API docs section
- Sign up / Log in forms
- Dashboard with upload area
- Receipt results view with JSON download
- API key management page
- Settings/profile page
